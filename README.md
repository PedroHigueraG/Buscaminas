# Buscaminas

Desarrollando el clasico juego de buscaminas.

## Componentes

- Programado en Java
- Patron MVC (Modelo, Vista, Controlador)
- Patron DAO
- Programación Orientada a Objetos
- Base de datos PostgreSQL
- IDE de IntelliJ IDEA
- Prototipado en figma
