# Buscaminas

Desarrollando el clasico juego de buscaminas en java, utilizando el IDE de intelliJEA IDEA.

## Componentes

- Programado en Java
- Patron MVC (Modelo, Vista, Controlador)
- Patron DAO
- Programación Orientada a Objetos
- Base de datos PostgreSQL
